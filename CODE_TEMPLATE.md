**Reversing a linked list**

```
def fn(head):
    curr = head
    prev = None
    while curr:
        next_node = curr.next
        curr.next = prev
        prev = curr
        curr = next_node

    return prev
```

def fn(arr, target):

    while left <= right:

        if arr[mid] == target:
